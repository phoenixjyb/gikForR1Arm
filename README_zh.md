# R1 机器人手臂 GIK 与抓取项目

## 项目简介
本项目基于 MATLAB，提供了 R1 机器人手臂的仿真与抓取运动规划流程。包含了稳健的工作空间采样、避障、自动瓶子放置等功能，确保抓取动作可行且安全。

## 主要功能
- **广义逆运动学（GIK）**：三阶段抓取（接近、抓取、闭合）。
- **工作空间采样**：使用统一的 `sampleReachableWorkspace` 函数。
- **自动瓶子放置**：若初始位置不可达，自动移动到可达位置。
- **与桌面等环境的避障**。
- **机器人、瓶子、桌面与轨迹的可视化**。
- **模块化代码**，便于扩展和适配其他物体或机器人。

## 环境配置
1. 克隆本仓库并在 MATLAB 中打开。
2. 确保所有文件在主目录下，URDF 文件在 `fineUrdfs/` 文件夹。
3. 运行 `createAndPlaceBottleTable.m` 生成瓶子和桌面对象。
4. 运行 `simpleGIKprocess_R1BodyAndMug.m` 仿真抓取流程并可视化结果。

## 主要脚本
- **`createAndPlaceBottleTable.m`**：
  - 加载机器人，采样工作空间，创建瓶子和桌面。
  - 检查瓶子的抓取位姿是否可达，不可达则自动移动并用 IK 验证。
- **`simpleGIKprocess_R1BodyAndMug.m`**：
  - 执行三阶段 GIK 流程（接近、抓取、闭合）。
  - 动画展示机器人、瓶子和桌面。
  - 打印并保存最终位姿。

## 工作空间采样
- `sampleReachableWorkspace(robot, eeName, numSamples, visualize)` 随机采样关节空间，计算末端执行器的可达空间。
- 支持连续关节与无效采样的鲁棒处理。
- 可选可视化。

## 自动瓶子放置
- 采样后，脚本检查瓶子的抓取位姿是否在任一采样点 5cm 内。
- 若不可达，自动移动到最近的采样点，并用 IK 验证。
- 仅在 IK 成功时更新瓶子，确保放置可行。

## 常见问题
- 若提示瓶子不可达，脚本会自动尝试移动。
- 请确保所有依赖文件（URDF、辅助函数）在正确位置。
- 如需适配新物体或机器人，请修改相关脚本和 URDF 路径。

## 扩展说明
- 可通过修改 `createBottleAndTable.m` 适配其他物体。
- 工作空间采样函数可用于任意机器人和末端执行器。
